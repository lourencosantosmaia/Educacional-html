Float Right

Clear Float: aprenda como usar a propriedade CSS Clear facilmente.


O clear float do CSS determina como os elementos flutuantes se comportam . Ambos float e clear são propriedades que andam de mãos dadas. Quando você flutua um elemento, permite que os elementos adjacentes tentem fluir em torno dele, o que pode resultar em layouts estranhos. Continue lendo para saber como evitar designs estranhos por meio da propriedade clear.


Além de envolver o texto em torno dos elementos, você pode usar flutuações para criar layouts da web . Além do mais, os floats são úteis ao lidar com pequenas instâncias de layout. Por exemplo, essa propriedade permite que você flutue uma pequena imagem de avatar enquanto permite que o texto ao seu redor reflua para acomodar quaisquer alterações de imagem.


left: Com o valor CSS float left, você faz um elemento flutuar à esquerda do contêiner nenhum: É o valor padrão que não faz flutuar os elementos. right: O valor CSS float right permite que um elemento flutue à direita do contêiner. herdar: permite que um elemento herde o valor flutuante do elemento pai. Nota: Float não suporta valores para a parte inferior, superior e central. Isso ocorre porque não é possível flutuar elementos para cima, para baixo ou para o centro de um contêiner.
